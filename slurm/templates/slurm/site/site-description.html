This is a web dashboard for the SLURM scheduler built with Django, Bootstrap, and pyslurm.

You can change this message and customize it for your site by editing ./slurm/templates/slurm/site/site-description.html. This is probably where you'd write an informative message about your cluster or organization or something like that. Check out the ./slurm/templates/slurm/site and ./slurm/static/slurm folders for additional customization options. You are encouraged to change whatever :)
